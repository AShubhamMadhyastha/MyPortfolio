TaskMaster

A web application for managing tasks, tracking productivity, and organizing due dates with a built-in calendar feature.

The app features a clean, user-friendly interface built with JavaScript, and is powered by Node.js on the backend. Color Picker App

A web application for selecting and managing colors, viewing color codes, and generating gradients with a built-in history and sharing features.

In this project, I developed a web-based color picker application that allows users to choose colors, view their corresponding color codes in multiple formats (Hex, RGB, HSL), and manage their color history. The app also includes features for creating gradients, switching between light and dark themes, simulating color blindness, and sharing colors via a unique URL. It has an intuitive, user-friendly interface designed with HTML, CSS, and JavaScript.

Technologies used: HTML, CSS, JavaScript

This project has given me the opportunity to refine my skills in front-end development and create a tool with real-world applications. It highlights my ability to build interactive, responsive web applications while handling user data and integrating multiple features for an enhanced user experience.

Advanced Calculator with Dark Mode

A fully functional calculator with support for basic and advanced mathematical operations, memory functions, and a light/dark theme toggle.

This project involved building a responsive web-based calculator that can perform a wide range of operations, from basic arithmetic to more advanced functions like square root, exponentiation, and logarithms. The app also includes a memory feature, allowing users to store and recall values. Additionally, users can toggle between light and dark themes to personalize their experience.

Technologies used: HTML, CSS, JavaScript.

Developing this calculator helped me improve my skills in DOM manipulation, event handling, and responsive design. It also gave me an opportunity to implement a light/dark mode feature and explore how user preferences can be easily managed with JavaScript. This project highlights my ability to build interactive and visually appealing web applications with clean and maintainable code.
